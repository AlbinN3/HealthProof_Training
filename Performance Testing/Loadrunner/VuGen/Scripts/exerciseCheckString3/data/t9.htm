CodeCharge Studio Demos and Examples
Rapid Web development tools are becoming increasingly important for a growing number of developers these days. CodeCharge Studio is being used in many innovative ways to create sophisticated Web applications. Take a look below at the professional-strength examples (many included with purchase) built using CodeCharge Studio. You won't believe how quickly you can start building better Web applications!
Complete Applications
  • YesSoftware Website
    Our own extensive, fully-functional Website was developed using CodeCharge Studio. It includes powerful product management and affiliate management features and a secure online store. It runs on Linux with Apache, MySQL and PHP.
  • CodeCharge Forums
    This powerful online discussion system has the advanced features found only in the leading community applications, such as integration with NNTP newsgroup server, advanced search, email notifications, spam filtering, and many more. It runs securely and efficiently on Linux with Apache, MySQL and PHP.
  • CodeCharge Support System
    The CodeCharge support site includes news/updates and documents depository and an online trouble ticket management system. It runs on Windows NT with ASP and MS Access.
  • CodeCharge Knowledge Base
    A user-friendly, searchable database-driven knowledge management solution that provides a centralized repository for company knowledge. Runs smoothly on Windows NT with ASP and MS Access.

    Note: the Knowledge Base application is available as an example, free of charge to all registered users.
